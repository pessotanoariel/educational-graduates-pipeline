# v1.1 — Publication & Monitoring Layer

## Overview

Version 1.1 completes the transition from a consolidation pipeline into a structured data product.

This release introduces operational publication capabilities, historical snapshots, monitoring workflows, and documentation improvements.

---

## Major Features

### Publication Layer

Implemented a dedicated publication layer to generate stable datasets for downstream systems.

Outputs:

```text
data/output/published/base_graduados_unificada_latest.csv
data/output/historical/base_graduados_unificada_YYYY-MM-DD.csv
```

Features:

* Operational dataset generation
* Legacy output compatibility
* Historical snapshots

---

### Monitoring Layer

Implemented historical quality monitoring and reporting.

Outputs:

* quality_summary.csv
* quality_history.csv
* enrichment_summary.csv
* null_distribution_report.csv
* duplicated_records_report.csv

Features:

* KPI tracking
* Historical monitoring
* Quality reporting
* Coverage analysis
